Install

You can install Sanelotto on any Linux-based system.

Dependencies

  • Python 3.x
  • git

Download

First of all, download latest version Sanelotto.

Version 1.8

Installing

For installing you must have right to write in /usr/local/bin/ and /usr/share/. If you do not have these right, use 'sudo'.

Start sanelotto_install.sh through terminal. After start console will display installing process. It does not take a lot of time.

If there was no errors, you can move to start.
